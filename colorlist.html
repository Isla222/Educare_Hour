RGB code

  • Specifies a parameter with three primary colours which are red, green and blue (red, green, blue).
  • Each parameter defines the intensity of the colour with a value between 0 and 255

According to the colour theory, you have to add red and blue in order to get purple, black and sky blue in order to get navy blue, and white and purple in order to get lavender. The same method of colour mixing goes to the RGB colour pallet, you have to adjust the value of colours you use between red, green and blue in order to get your desired colour.

For example-
If we set red to the largest value which is 255, you cannot set the other two values to their largest, since they will clash, and the same goes for the other colours
• RGB(255,0,0) for the most intense red
• RGB(0,255,0) for the most intense green
• RGB(0,0,255) for the most intense blue
And you have to adjust the parameter’s values to get the colour you want



HEX code

  • Hexadecimal
  • Specifies with #rrggbb (r=red, g=green and b=blue)
  • Integers between 00 and ff, represent the intensity of the colour; ff being the most intense

#ff0000 is the reddest red of the hexadecimal colour code because the RR's value is set to the most intense and the GG and BB are set to zeros
