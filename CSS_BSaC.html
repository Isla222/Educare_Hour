CSS Cascading Style Sheets
CSS is the language for describing the presentation of Web pages, including colors, layout, and fonts.












How to add CSS

There are 3 WAYS for style of CSS. We can use whatever we prefer.

Inline Style (Using style attribute)

We have to use that style in the tag with as the attribute. Advantage is we can see the style for only one tag.


Inline Style (Using style attribute)

We have to use that style in the head seaction with as the style tag. Advantage is we can see clearly the codes.


External Style (Using the another css file and link together)

We have to use that style as the spreating the file. Advantage is we can see clearly the codes in the another file but we link each file in the head seaction.


rel=relation
href=hypertext reference (file name)

Basic CSS

  1. Padding (space between border and letter)
    {border: 2px solid black; padding: 30px;}

  2. Margin (space between outside of border)
    {border: 2px solid black; margin: 30px;}
    Shortcut way to describe about the border

  3. Color (color of the text)
    {color: red;}
    Color is vale and red is property and the we can choose the color as we wish

  4. Text-align (where the text gonna take place) center,left,right
    {text-align: center;}
    Text align is the code that can adjust the position of the test that we wanna put



  5. CSS Comment
    /* comment */
    If we do comment on something, we have to use /* smth*/ because that code is the comment code. In that code, we can put whatever we like and it won’t be shown default view.

CSS Universal Selector (*)

*{ …… } [*all the elements are affected*]
When we wanna do the same style, we have to use that universal selector.


CSS Grouping Selector

H1,h2,p,...{ }
If we wanna get affect for h1 and p, we have to use h1,p then style code.
