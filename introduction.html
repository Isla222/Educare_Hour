HTML stands for Hyper Text Markup Language and it is the standard markup language for creating Web pages. HTML describes the structure of a Web page, and consists of a series of elements. HTML elements tell the browser how to display the content and label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

How the web works

Clients and servers

Computers connected to the internet are called clients and servers.

Clients

Clients are the typical web user’s internet-connected devices ( for example, computers being connected to the WI-FI, phones being connected to mobile networks) and web-accessing software available on those devices (usually a web browser like Chrome or Microsoft Edge)

Servers

Servers are computers that store web pages, sites, or apps. When a client wants to access the web, a copy of the web is downloaded from the server onto the client to be displayed in the user’s browser.







What happens when you type a web address into your browser?

  1. The browser goes into the DNS server (an address book for websites) and finds the actual place of the server that you typed in.
  2. The browser sends an HTTP request to the server, asking it to send a copy of the website to the user. It is sent across your internet connection using TCP/IP (Transmission Control Protocol / Internet Protocol are communication protocols that define how data should travel across the internet).
  3. The server looks for the file and responds with an HTTP response; a “200 OK”, if you can look at the website or a “404 Not Found” if you can’t.
  4. If you can look at the website, it prepares the images, style sheets and scripts and displays them to you.
