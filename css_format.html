Formatting Text

Formatting Text

→ h1{font-size:1.5em;} 1em is equal to the current font size.The default text size in browsers is 16px.So, the default size of 1em is 16px.

→ h1{font-size: 150%;} /*150% of 16 =24*/ The solution that works in all browsers, is to set a default fontsize in percent for the < body > element.
→ /*(xx-small , -small, small, medium, large,x-large,xx-large)*)/ h1 {font-size: x-large;}
→ /*larger or smaller than the surrounding text*/ strong {font-size:larger;}


Font Weight , Font-Style

Font Weight

→ The font-weight property specifies the weight of font:
→ font-weight : normal | bold | bolder | lighter | 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900 | inherit

Font-Style

→ The Font-Style property used to specify italic text.
→ Font-Style : normal | italic | oblique | inherit


Variant , Text-indent

Variant

→ The font-variants specifies whether or not a text should be displayed in a small-caps font.
→ font-variant :normal | small-caps | inherit

Text-indent

→ The text-indent property is used to specify the indentation of the first line of a text:
→ text-indent :length measurement | percentage | inherit


Text allignment and decoration, Text Transform

Text allignment and decoration

→ The text-allign property is used to set the horizontal alignment of a text. A text can be left or right aligned , centered , or justified.
→ The text-decoration property is a shortand property for:
text-decoration-line(required)
text-decoration-color(optional)
text-decoration-style(optional)
text-decoration-thickness(optional)
→ text-align : left | right | center | justify | inherit
→ text-decoration: none | underline | overline | linethrough

Text Transform

→ The text-transform property is used to specify upperspace and lowercase letters in a text.
→ text-transform :none | capitalize | lowercase | uppercase | inherit


Spacing , Text Shadow

Spacing

→ The letter-spacing property is uesed to specify the space between the characters in a text.
The word spacing property is used to specify the space between the words in a text.
→ letter-spacing : length measurement | normal | inherit
→ letter-spacing : length measurement | normal | inherit


Text Shadow

→ The text shadow property adds shadow to text.
→ text shadow : horizontal offset | vertical offset | blur radius | color | none
