HTML Attributes

  • All HTML elements can have attributes.

  • Provides additional information about the elements.

  • Always in the start tag.

  • Usually come in name/value pairs like name=‟value”



href=‟value” < a href=‟document location” > Specifies the location of the link goes to
src=‟value” < img src=‟location of an image” > Specifies the source of something
width=‟value”

height=‟value”		 < div width=‟50px” height=‟50px” > Specifies the width and height of something
‟alt=value” < img src=‟location of an image” alt=‟an image” > Specifies an alternate text for something if the actual source attribute doesn’t work
style=‟value” < p style=‟color:red;” > Specifies style of an element
lang=‟smth” < html lang=‟en” > Used for making text boldSpecifies the language of the WEB page
class=‟name” < p class=‟paragraph1” > Specifies one or more class names for an HTML element
id=‟name” < p id=‟paragraph1” > Specifies an ID for an HTML element
