Anatomy of an HTML document












HTML META TAG

META tag

→ data about data

→ holds information about other information, but the information is in the form of raw data
→ provide Metadata of your HTML page
→ Invisible on the web page
→ seen in the source code of the HTML page
→ convey to the servers some information about what the web page is about and what keywords it has



HTML ROOT ELEMENT

ROOT element

→ Represents the root of an HTML document
→ All other elements are the descendants of this element



HTML HEADING ELEMENT

HEADING element

→ Contains machine readable information (metadata) about the document



HTML BODY ELEMENT

BODY element

→ Represents the content of an HTML document
→ All of the content of an HTML content goes between the opening tag and the closing tag of the body element



Anatomy of an HTML tag

an HTML tag

→ If you want to add some attribute in the tags, you have to make sure there is space between the tag element and the attribute name. (you can’t add attributes in closing tags)
→ You have to put attribute value in double or single quotes
→ Must close the content container tags
